Команда: own За допомогою цієї команди ви можете створювати власні команди і налаштовувати їх. Для початку створіть власну команду: (ваша команда не повинна містити пробілів) own <ваша команда> <ваша відповідь> Тепер ви можете налаштувати вашу команду. Затримка часу в секундах, для відповіді: own <1-60> <команда зі списку> Відповідь на вашу команду у вбудованому повідомленні: own <команда зі списку> Встановіть колір смужки вбудованого повідомлення: own color 5c68ee <команда зі списку> Вимкніть префікс: own <команда зі списку> Тепер ваша команда - це окреме слово, яке знаходиться всередині речення. own <команда зі списку> Тепер ваша команда міститься будь-де всередині речення. Увімкніть префікс: own <команда зі списку> Змініть відповідь вашої команди: own edit <ваша команда> <ваша відповідь> Видаліть вашу команду: own <ваша команда>
